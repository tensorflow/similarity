post /add
add item(s) to the index (addPost)
By passing in an array of items, you can add them to the indexer

Query parameters

items (required)
Query Parameter — array of items that should be added to the index

Return type

array[Integer]

Example data

Content-Type: application/json
[ 45, 46, 47 ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

201

item(s) created

400

invalid input, object invalid
Up 
delete /delete
delete item(s) from the index (deleteDelete)
By passing in an array of ids, you can delete the corresponding items from the indexer

Query parameters

ids (required)
Query Parameter — the indeces of the items in the dataset that should be removed

Responses

200

item(s) deleted

400

items do not exist

get /lookupEmbeddings
searches for the nearest neighbors (lookupEmbeddingsGet)
By passing in a list of embeddings, you can search for the nearest neighbors and their associated data in the index

Query parameters

items (required)
Query Parameter — array of embeddings for which to find the nearest neighbors
num_neighbors (required)
Query Parameter — number of neighbors that should be returned for each embedding

Return type

array[Neighbor]

Example data

Content-Type: application/json
[ {
  "data" : [ 6.027456183070403, 6.027456183070403 ],
  "distance" : 1.4658129805029452,
  "id" : 0,
  "label" : 5
}, {
  "data" : [ 4.027456183070403, 3.027456183070403 ],
  "distance" : 1.8954684,
  "id" : 5,
  "label" : 5
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

found nearest neighbors

404

bad request

500

internal server error

Info Up

number_embeddings
Integer The number of embeddings in the index
example: 60000
serving_directory
String The directory that is being served by the indexer
example: /MNIST
embedding_size (optional)
Integer The size of the embeddings stored in the indexer
example: 18

Item Up

An Item contains an example, its label and the original example if different from example
example
array[BigDecimal] example that should be added to the index
example: [1,12,435,9,234,2]
label
BigDecimal label associated with the example
example: 12
original_example (optional)
array[String] The original data if different from example format: bytes
example: ["<START>","This","is","an","example","<STOP>"]

Metrics Up

number_lookups
Integer The number of lookups that were performed by the indexer
example: 527
average_time
BigDecimal The average time to query the index

Neighbor Up

id
Integer The index of the item in the dataset
data
array[BigDecimal] The index of the item in the dataset
distance
BigDecimal The distance between the datapoints in embedded space
label
Integer The label of the item
