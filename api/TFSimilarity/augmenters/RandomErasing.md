# TFSimilarity.augmenters.RandomErasing





Applies RandomErasing to a single image.

```python
TFSimilarity.augmenters.RandomErasing(
    probability: float = 0.25,
    min_area: float = 0.02,
    max_area: float = (1 / 3),
    min_aspect: float = 0.3,
    max_aspect=None,
    min_count=1,
    max_count=1,
    trials=10
)
```
